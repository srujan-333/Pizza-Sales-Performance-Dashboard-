# Pizza-Sales-Performance-Dashboard-

# Project Overview
This project demonstrates an end-to-end data analysis workflow using **SQL** for data querying and **Power BI** for interactive dashboard creation.  
The goal is to extract meaningful business insights from raw data and present them in a visually intuitive format.

# Tools & Technologies
- SQL (Data querying & transformations)
- Microsoft Power BI (Dashboard & visualization)
- Excel / CSV (Data source).

# Key Features
- Data cleaning and transformation using SQL
- KPI metrics such as total orders, revenue, and trends
- Interactive Power BI dashboard with slicers and filters
- Insight-driven visualizations for decision making.

# Insights Generated
- Trend analysis over time
- Category-wise and region-wise performance
- Identification of high-performing segments.
